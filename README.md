# AI Website Generator for Local Businesses

## Business Chosen
Bright Future Coaching Institute, Ayodhya

## Objective
To build a complete, real, runnable website using prompt engineering and AI website builders.

## Tools Used
- Framer AI Website Builder
- ChatGPT (for prompt design)

## Prompt Strategy
A structured master prompt was designed to:
- Generate website layout
- Create conversion-focused content
- Adapt tone for a coaching institute
- Produce publish-ready webpages

## Outcome
A fully functional website was generated using AI prompts and published using Framer.
